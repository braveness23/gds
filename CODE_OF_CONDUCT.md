# Code of Conduct

## Our Standards

This project is a collaborative technical community. We expect all participants to:

- Be respectful and constructive in discussions and code reviews
- Focus feedback on ideas and code, not on people
- Welcome contributors of all experience levels
- Accept that others may have different approaches — discuss tradeoffs, don't dismiss

## Unacceptable Behavior

- Personal attacks or insults
- Deliberate intimidation or harassment
- Sustained disruption of discussions

## Scope

This applies to all project spaces: GitHub issues, pull requests, discussions, and any other venue where contributors represent this project.

## Enforcement

Project maintainers may remove, edit, or reject contributions that violate these standards, and may temporarily or permanently ban contributors for repeated or severe violations.

Report issues by opening a GitHub issue marked **[conduct]** or by contacting a maintainer directly.

This project follows the spirit of the [Contributor Covenant](https://www.contributor-covenant.org/).
